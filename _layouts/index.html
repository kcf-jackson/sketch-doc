What is Sketch?

- Sketch is an R package for creating static, animated or interactive sketches and illustrations embeddable in R Markdown documents. - It implements an R-to-JavaScript transpiler and enables users to write JavaScript applications using the syntax of R. - Designed for scientists, journalists, artists, business users.
